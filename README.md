# Climate-Change-Analysis-and-Prediction
The proposed system successfully analyzes climate data and predicts key indicators such as global temperature, rainfall, and air quality index (AQI) using data analytics, machine learning, time-series, and deep learning techniques. Through Exploratory Data Analysis (EDA), important patterns and trends were identified, helping to understand the impact of various environmental factors on climate change.
The machine learning, ARIMA, and LSTM models demonstrated effective performance in forecasting climate conditions, enabling better planning and environmental decision-making. However, challenges such as missing values, data inconsistency, and limited availability of real-time data were encountered during implementation.
Overall, the system proves to be a valuable tool for climate analysis by improving prediction accuracy, supporting data-driven insights, and contributing toward environmental sustainability and climate monitoring.
